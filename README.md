### 🗳️ AmarVote — Secure Online Voting System

- End-to-end **verifiable e-voting** using **Microsoft ElectionGuard SDK**
- **Post-quantum encryption (ML-KEM 1024)** for guardian key protection
- **Blockchain-backed auditing** (Ganache) for immutable ballots
- **Spring Boot (JWT)** backend + **React** frontend
- Deployed on **Microsoft Azure**
